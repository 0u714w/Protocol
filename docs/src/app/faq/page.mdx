# Frequently Asked Questions

## General Questions

### What is Surfer Protocol?
Surfer Protocol is a decentralized data management protocol designed for secure and efficient data operations.

### Is Surfer Protocol open source?
Yes, our core protocol is open source and available on GitHub.

### How can I get started?
Start by installing our [Desktop application](/desktop/installation) or exploring our [Python SDK](/python-sdk).

## Technical Questions

### What platforms are supported?
We currently support major desktop operating systems and provide a Python SDK for developers.

### How secure is the protocol?
We implement industry-standard encryption and security measures, with regular security audits.
