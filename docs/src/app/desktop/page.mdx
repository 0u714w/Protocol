# Desktop Application

The Surfer Protocol Desktop application is a cross-platform application that allows you to easily export your data.

## Supported Platforms 

The Desktop Application currently supports the following platforms:

- [Twitter](/desktop/platforms/twitter)
- [ChatGPT](/desktop/platforms/chatgpt)
- [Notion](/desktop/platforms/notion)
- [Gmail](/desktop/platforms/gmail)
- [iMessage](/desktop/platforms/imessage)
- [LinkedIn](/desktop/platforms/linkedin)

## Schema Format

The exported data for the supported platforms is in JSON format like this:

```json
{
  "company": "Company Name",
  "name": "Platform Name",
  "runID": "platform-run-id",
  "timestamp": 1234567890,
  "content": [ 
    {
      "key_1": "value_1",
      "key_2": "value_2",
      "key_3": "value_3"
    },
    ...
  ]
}
```
