# Contributing to Surfer Protocol

We welcome contributions from the community! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a new branch
4. Make your changes
5. Submit a pull request

## Where to Contribute

- Adding platforms for export
- Documentation improvements
- Building applications with Surfer Protocol
- Bug fixes
